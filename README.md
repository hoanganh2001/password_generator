# PasswordGenerator

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 16.2.5.

## Build

- chạy terminal
- gõ 'npm i'
- xong r thì chạy 'ng serve'
- auto nhận link là localhost:4200
- push file excel lên r nhấn run là tự tải file về(file phải là example giống file excel mấu ở ngoài)

## Note

- Build thành file exe sau
- Mỗi Field hiện tại chỉ chứa được 1 data VD: job: it chưa được kiểu job: it,sale
- birthday bắt buộc ô đấy phải được cell format dạng DAte với MM/DD/YYYY
